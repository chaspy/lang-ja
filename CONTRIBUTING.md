# How to contribute


## 注意事項

closeされたissueやPRおよびチェンジセットへのコメントは、見逃される(無視される)
可能性があります。どんな些細な事でも気になることがあったら遠慮せず、大事だと思
うことならさらに積極的に、新たにissueを作ってください。

表語            |例
----------------|-----------------------------------------------------------
